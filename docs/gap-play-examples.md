# Gap play examples

A ready-made list of **stocks in play** to practice on. Each row is a real
ticker-day with a large opening gap and heavy, liquid premarket volume — exactly
the kind of session worth replaying on the tape. Use it to pick a **Symbol** +
**Date** for the [download form](#downloading-data), then [load the day](#loading-a-day)
and read the flow.

## How this list was built

These are gap plays from **2023-01-03 to 2026-05-13**, screened so every name is
big and liquid enough to have real order flow:

- **Gap ≥ 5%** at the open (up or down).
- **Average dollar volume ≥ $100M** — large, liquid names only.
- **Premarket dollar volume ≥ 50%** of the 4-week average daily dollar volume —
  the gap came with genuine premarket participation, not a thin print.
- **Tradable only** (common stock + ADRs; `CS`/`ADRC`).
- **Forward-looking ATR-based filter** to screen out buyouts/mergers — a
  cash-deal close has a tiny realized range relative to the gap, so a minimum
  ATR ratio (0.55) drops those "dead" post-deal tapes while keeping live movers.

The columns mirror the screener output:

| Column | Meaning |
|---|---|
| **Gap** | Open vs. prior close, %. |
| **RVOL** | Relative volume vs. the stock's own recent average. |
| **PM%adv** | Premarket dollar volume as a % of the 4-week average daily dollar volume. |
| **Score** | Composite rank (higher = stronger setup). |
| **ATR ratio** | Realized range vs. expected ATR — the buyout/merger filter. |

> **Tip:** the highest-**Score** rows (e.g. `BYND` 2025-10-21, `VKTX` 2024-02-27,
> `OPEN` 2025-07-21, `GME`/`AMC` May 2024) are the most dramatic tapes to start with.
