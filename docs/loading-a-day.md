# Loading a day

The **Library** is the home screen. It lists every market day you've downloaded and lets you open one to trade.

## The library list

Each downloaded day is a row showing:

- the **symbol** and **date** (e.g. `EOSE   2026-05-13`),
- how many venues and how big it is (e.g. *"15 venues · 342 MB"*),
- an **Open** button.

If you haven't downloaded anything yet, you'll see *"No downloaded days yet. Use the form below to download one."* — head to [Downloading market data](#downloading-data).

## Opening a day

Click **Open** on a row. TapeSim builds the session from the recorded files — you'll see a brief **Loading…** message with an elapsed timer (a busy day has millions of records, so this can take a few seconds). When it's ready, the [trading screen](#the-trading-screen) appears, positioned at the start of the session.

## Going back

In the trading screen, click **← Library** (top-left) to return to the list and pick another day.

If you have an **open position** when you leave, TapeSim asks you to confirm first — leaving discards that simulated session. Opening a new day always starts fresh (the previous session is closed and its memory released, so you're never holding two days at once).
